# כלל שיוך פרקים

כל פרק או PDF חייב להיות משויך למבנה הבא:

רמה → שכבה / יחידות → תחום / שאלון → פרק

דוגמה:

חטיבת ביניים → כיתה ז → אלגברה → משוואות

או

בגרות → 5 יחידות → שאלון 571 → חקירת פונקציה


## מדיניות PDF (קריטי)
- אין לייבא/להעתיק PDFs אוטומטית מה-Downloads או מכל מקום אחר.
- רק קבצים שיוכנסו ידנית לנתיב `pdf/` בתוך הריפו ייכללו בספר.
- כל שינוי בתכולת `pdf/` חייב להיות החלטה מפורשת ולבוא עם commit.
