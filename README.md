# DigitalBook — ספר דיגיטלי (PDF)

אתר סטטי להצגת קבצי PDF מתוך הריפו כספר דיגיטלי בעברית (RTL), עם:

- תוכן עניינים גלובלי (עץ תיקיות)
- תוכן עניינים פנימי לפרק (Outline של PDF כשקיים)
- ניווט עמודים ופרקים
- חיפוש לפי שם פרק + חיפוש בתוך הפרק
- תמיכה בהתקנה כ־PWA

## הוספת PDFs

1. הוסיפו קבצי PDF לתיקייה `pdf/` (אפשר גם בתיקיות משנה לפי המבנה ב־RULES.md).
2. בצעו `git commit` + `git push`.
3. GitHub Actions יעדכן אוטומטית את `site/generated/toc.json`, יעתיק PDFs ל־`site/pdf/`, ויפרסם ל־GitHub Pages.
